✨ Features

✅ WhatsApp Multi-Device Support
✅ Plugin Command System
✅ Auto React Messages
✅ Fast & Lightweight
✅ Easy Deployment

---

📦 Installation

Clone the repository:

git clone https://github.com/Dexsam07/Dex-mini-malvin-king
cd Dex-mini-malvin-king

Install dependencies:

npm install

Start the bot:

node index.js

---

⚙️ Configuration

Edit config.js

Example:

module.exports = {

OWNER_NUMBER: "917XXXXXXXXX",

BOT_NAME: "Dex Mini Malvin King",

PREFIX: ".",

AUTO_REACT: "❤️"

}

---

🧾 Commands

Command| Description
.ping| Check bot speed
.alive| Bot status
.menu| Show command list
.owner| Owner contact
